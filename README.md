# 郑州航院校园AI助手（小航）
## 一、项目简介
本项目基于Python + Streamlit Web框架 + 大模型API开发面向郑州航空工业管理学院师生的智能问答系统。
系统内置新生、在校生、教师三种身份角色，差异化输出回答风格；加载本地校园知识库约束大模型输出，杜绝AI幻觉编造信息；配套分类推荐标签页、多轮上下文对话、问答历史持久展示、对话记录导出txt、一键清空会话、输入空值拦截、接口超时自动重试等完整交互功能。
适用于新生报到咨询、在校师生办事指引、教师行政业务咨询，简化校园事务咨询流程。

## 二、开发环境
1. Python版本：3.10 及以上
2. 核心依赖库：
   - streamlit：Web可视化页面框架
   - requests：HTTP网络请求库，调用大模型API
3. 外部服务：硅基流动大模型对话API
4. 版本管理工具：Git
